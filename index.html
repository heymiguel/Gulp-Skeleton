Gulp-Skeleton

A little workflow using Gulp, Skeleton and more

Prerequisites
Need to have NodeJS and npm installed
More info here
Need to have Git installed
More info here
Installation
Download the project
git clone https://github.com/MrDragonXM15/Gulp-Skeleton.git PROJECTNAME
Go into the folder
cd PROJECTNAME
Install all dependencies
npm install
Usage
Start a live server
gulp
This launches a server to have a live preview of the changes made.
Publish your website
You just need to publish your /dist/ folder
Informations
This project uses Jade/Pug
You must know how to use Pug. To learn how to use it, you can watch this playlist or go to this website
This project uses sass
If you want add personnal style, you need to add them to a file called FILENAME.scss. You can add css or scss, it's not a problem.
This project uses ESLint
You can learn more about it here
Minification
The minification is done automatically when saving any file.
